# 🚀 DEVCONF 2026 - Conference Landing Page

A modern conference landing page built using **HTML5** and **CSS3**. This project showcases a fictional developer conference with featured speakers, ticket pricing, and event schedule.

## 🌐 Live Demo

🔗 https://sabermahmud.github.io/a1-devConf-2026/



---

## ✨ Features

- Full-screen hero section with background image
- Featured speakers section
- Conference ticket pricing cards
- Event schedule section
- Modern footer
- Google Fonts integration
- Smooth hover effects and transitions
- Clean and organized code structure

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts (Inter)
- Font Awesome
